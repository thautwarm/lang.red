# Language-1 Fairy

```
c = "license"
f(head::tail)([a, b, ~c])
        when a==b then head
g(a:int, 1)(b:int, 2)(c:int, 3):int = 
        when a == b then c
        when b == c then a
        when a == c then b
        otherwise   then summary
    where summary = a+b+c
```
